# Agency System

Bu klasör, mind-agent prototipinden **tamamen izole** yeni bir geliştirme alanıdır.

## İzolasyon Kuralı

- `agency/` dışına hiçbir dosya **yazılmaz**
- `src/`, `tests/`, `docs/` klasörlerine **dokunulmaz**
- Geliştirme bitince tek bir merge ile ana sisteme bağlanır

## İçerik

- `docs/` — Mimari kararlar, departman tanımları, akışlar
- `src/` — Yeni sistem kodu (hazır olunca)
- `tests/` — Yeni sistem testleri (hazır olunca)

## Durum

⏳ Mimari tasarım aşaması
