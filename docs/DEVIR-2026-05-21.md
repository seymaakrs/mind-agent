## 🎓 Kavramlar (Şeyma için not)

- **Branch protection ruleset** = main branch'in "kapısı"; PR olmadan direkt yazılmaz. Bugün açıldı.
- **Cloud Run Job** ≠ Cloud Run Service. Service uzun süre ayakta (FastAPI), Job kısa süre çalışıp exit eder (cron tetikli batch işi).
- **RUN_ONCE=true env** = runner kodunun "sonsuz loop yerine bir tick yap, çık" demesi. Cloud Run Job için şart.
- **Deprecation alias** = eski ismi tamamen silmek yerine yeni isme bağlamak. Geriye dönük uyum. Bir sonraki major versiyonda silinir.
- **Squash merge** = bir PR'daki tüm commit'leri tek commit'e çevirip main'e koymak. Tarih temiz kalır.
- **Bypass list** = ruleset'i atlayabilecek roller. "Repository admin" eklendi → sen bypass edebilirsin (kendi PR'ını approve etme gereksiz).
- **HMAC signature** = webhook'a gelen istek gerçekten karşı taraftan mı geliyor doğrulaması. `ZERNIO_WEBHOOK_SECRET` ile.
